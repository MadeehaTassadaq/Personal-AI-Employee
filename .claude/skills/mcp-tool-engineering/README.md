# MCP Tool Engineering Skill

This skill provides comprehensive guidance for designing, validating, and implementing Model Context Protocol (MCP) tools used by AI agents to interact with application backends.

## Files Included

- `SKILL.md` - Main skill definition with all requirements and guidelines
- `references/patterns.md` - Implementation patterns and best practices
- `references/example_tool_specs.json` - Example tool specifications for validation
- `scripts/mcp_validator.py` - Validation script to check tool compliance
- `assets/quick_reference.md` - Quick reference guide for users

## Features

- Validates MCP tools for statelessness, determinism, and security
- Provides structured error handling patterns
- Includes example implementations for common operations
- Offers validation checklist and anti-pattern identification
- Contains reusable validation script for automated checking

## Usage

This skill should be activated when:
- Creating MCP tools for backend integration
- Designing tool interfaces for AI agents
- Validating stateless execution requirements
- Implementing database persistence for tools
- Ensuring agent-safe tool behavior
