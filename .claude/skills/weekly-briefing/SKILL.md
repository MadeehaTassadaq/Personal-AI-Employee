# Weekly Briefing

Generate executive summaries of weekly activity.

## Data Sources

- `$VAULT_PATH/Dashboard.md` - Current status
- `$VAULT_PATH/Done/` - Completed tasks (filter by date)
- `$VAULT_PATH/Needs_Action/` - In-progress tasks
- `$VAULT_PATH/Pending_Approval/` - Awaiting approval
- `$VAULT_PATH/Logs/` - Activity history

## Output Format

```markdown
# Weekly CEO Briefing - {{DATE_RANGE}}

## Executive Summary
{{2-3 sentence overview of the week}}

## Completed This Week
- Task 1 (completed DATE)
- Task 2 (completed DATE)

## In Progress
- Task 3 (started DATE, current status)

## Pending Approval
- Action 1 (waiting since DATE)

## Bottlenecks
- {{Any blocked or delayed items}}

## Recommendations
- {{Proactive suggestions, max 3}}

---
*Generated by Digital FTE on {{GENERATION_DATE}}*
```

## Workflow

1. **Determine date range**: Default is last 7 days
2. **Scan Done folder**: Find tasks completed in range
3. **Scan active folders**: Get current Needs_Action and Pending_Approval
4. **Read logs**: Extract activity patterns
5. **Identify issues**: Find delays, blockers, long-pending approvals
6. **Generate report**: Create briefing in specified format
7. **Save report**: Write to `$VAULT_PATH/Plans/weekly-briefing-{{DATE}}.md`

## Rules

- Factual only: Never fabricate metrics or data
- Highlight delays honestly
- Keep recommendations actionable and specific
- Tone: Professional, concise, CEO-appropriate
- Max 3 recommendations per briefing
- Include generation timestamp
